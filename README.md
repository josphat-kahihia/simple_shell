# Simple Shell Project

## Overview

This project is a simple implementation of a Unix-like shell in C. The shell supports basic command execution, I/O redirection, and piping. It provides a command-line interface for users to interact with the operating system.

## Features

- Command execution: Run external commands and display their output.
- I/O redirection: Redirect input and output for commands.
- Piping: Chain commands together using pipes.
- Background execution: Execute commands in the background.
- Signal handling: Handle signals such as Ctrl+C for graceful termination.

## Getting Started

### Prerequisites

- C compiler (e.g., GCC)
- Unix-like operating system (Linux, macOS)
